# DinoNuggets-Chatbot
Dino Nuggets is a chatbot using a knowledge base document using facts about dinosaurs. The front end is html/css/javascript. While the back end is python. Flask was used to connect our front end to back end. A RAG (retrieval augmented genertation) was followed to create the Dino Nuggets Chatbot. 

## Installation Requirements
* run```pip install -r requirements.txt``` from your IDE terminal to install all of the required dependencies at once
* you could also install them one by one if you prefer
